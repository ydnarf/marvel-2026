# Marvel Studios — Estrenos 2026

Landing page con todos los estrenos de Marvel Studios para 2026 y más allá.

🔴 **[Ver en vivo → marvel2026app.netlify.app](https://marvel2026app.netlify.app/)**

---

## Contenido

- Timeline interactivo con todos los estrenos de 2026 y 2027
- Cuenta regresiva en tiempo real hacia Avengers: Doomsday
- Películas destacadas: Spider-Man: Brand New Day y Avengers: Doomsday
- Series Disney+: Wonder Man, Daredevil Born Again T2, The Punisher, X-Men '97 T2, VisionQuest
- Modal de trailers con reproductor de YouTube
- Sección "Más allá de 2026": Secret Wars, Armor Wars, Nova, Shang-Chi 2, Blade

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- Fuentes: Bebas Neue + Inter (Google Fonts)
- Animaciones con Canvas API e IntersectionObserver
